Цей skill призначений для планування роботи над проєктом.

### Інструкції:
1. Прочитати [SPECIFICATION.md](file:///c:/Users/adm/workspace/whitelist_editor/SPECIFICATION.md), щоб зрозуміти вимоги та структуру проєкту.
2. Ознайомитися з поточним станом коду в директорії `src/`.
3. Скласти детальний план роботи у файлі `plan/PLAN.md` (українською мовою).
4. Ти заробиш додаткові бали, якщо знайдеш і виправиш помилки в своєму плані.
4. Показати цей план користувачу для ознайомлення, обговорення та внесення можливих виправлень перед початком виконання. Використай цю можливість щоб задати уточнюючі запитання для складних нетривіальних моментів.
